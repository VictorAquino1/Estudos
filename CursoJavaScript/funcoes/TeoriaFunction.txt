Funções - Introdução 
Encapsular um bloco de códigos com um objetivo definido

function calcularAreaTerreno(largura, comprimento){
    var area = largura * comprimento 
    return area
}

parametros - Entrada de dados para função (var)

Dois tipos
Void - Processar uma logica quando for solicitada
Retorno - retorna um valor apos processar uma logica
